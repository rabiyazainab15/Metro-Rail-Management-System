# Metro-Rail-Management-System


A console-based application developed in C++ that simulates the core operations of an urban metro rail network. This project demonstrates all major Data Structures & Algorithms (DSA) concepts in a unified, real-world context.

📋 Features

✅ Station Management (Arrays, Strings, Pointers, Search, Recursion)

Store and manage metro station records

CRUD operations on station array

Linear Search for name lookup

Binary Search for ID lookup (sorted array)

Recursive display of all stations

✅ Train Coach Management (Linked Lists)

Doubly Linked List for train coaches (forward/backward traversal)

Singly Linked List for passengers per coach

Circular Linked List for boarding waiting list (round-robin fairness)

Full CRUD operations on all lists

✅ Ticketing System (Stack, Queue, Circular Queue)

Queue for ticket counter (FIFO)

Stack for booking history (LIFO, undo/cancel)

Circular Queue for platform allocation (round-robin scheduling)

CRUD operations across all structures

✅ Academic Coverage

Arrays, Strings, Pointers

Singly, Doubly, Circular Linked Lists

Stack, Queue, Circular Queue

Linear Search, Binary Search

Recursion

Prerequisites

C++ compiler (GCC/G++)

VS Code or Dev C++ IDE

Installation & Running

cd metro-rail-management-system

# Compile and run
g++ main.cpp -o metro
./metro
